empty :: Element

empty(element) -> Boolean

 

Tests whether element is empty (i.e. contains only whitespace).

 

Examples

 

<div id="wallet">     </div>

<div id="cart">full!</div>

 

$('wallet').empty();

// -> true

 

$('cart').empty();

// -> false
